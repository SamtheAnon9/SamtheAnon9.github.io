Main Projects

On this page, I’ve included released projects that I was on for the majority of their development. For various other projects that I’ve assisted on for smaller periods of time, see the ‘Other Contributions’ page here.

Rise of the TMNT : Ninja Run

Description: A runner game for mobile, in which players control the Ninja Turtles, and have to defeat various enemies on a quest to recover their missing pizza

Tools Used: Unity with C#, Javascript backend with Playfab

Engineers: Team size varied between 1-2 engineers

Involvement: I was heavily involved with all of the features in the project, and was particularly involved in the creation of:

  • The kit system for loading the environment, and the editor tools used by artists and designers to create levels
  • The Playfab server backend and meta (currencies, levels, etc.)
  • The boss fights

Jellynauts

Description: A multiplayer brawler/.io game in which players absorb smaller blobs to become more powerful, and fight other players with a wide range of different abilities

Tools Used: Unity with C#, Javascript backend with Playfab

Engineers: Team size varied between 1-2 engineers

Involvement: I was the only engineer on the project for most of its development, with a few months in the middle of working with a senior engineer. Worked on everything, but some systems that I was particularly responsible for are:

  • The networked multiplayer
  • The AI for bot players
  • The meta/economy and Playfab backend

Dumb Ways to Dash

Description: A physics-based runner game in soft launch for mobile in which you have to beat other characters in a race to the end while avoiding various traps.

Tools Used: Unity with C#

Engineers: 1 engineer

Involvement: I was the only engineer on this project up until shortly after its initial soft launch, at which point I stopped working on the project. I created all of the systems used in that version, some particularly noteworthy systems were:

  • AI for other characters
  • Physics-based movement system
  • Ragdoll recovery system

Jumanji : Epic Run

Description: A 3-lane runner game currently in soft launch for mobile in which you run through the world of Jumanji

Tools Used: Unity with C#

Engineers: Team size varied between 2-3 engineers

Involvement: I joined this project a few months after it was started, and was mostly involved in helping out the other developers and fixing bugs. Some particular focuses were:

  • Changes to asset loading systems
  • Improvements and fixes to the kit system
  • Adding the FTUE/tutorial

Cars: Lightning League

Description: A racing/3-lane runner game for mobile in which you race as characters from Disney Pixar’s Cars movies

Tools Used: Unity with C#

Engineers: Team size varied between 2-3 engineers

Involvement: I joined this project about halfway through its development, was mostly involved in non-gameplay tasks. These included:

  • Creating a system for downloading the remainder of the game’s assets over-the-air while players play through the tutorial
  • Systems for the meta/economy
  • Implementing various plugins for things such as ads, analytics, and in-app purchases
